# Narrative Control Layer - Technical Design

## Architecture Overview

The narrative control layer adds three orthogonal systems on top of the existing novel pipeline:

```
┌─────────────────────────────────────────────────────────────┐
│                    Novel Pipeline (Existing)                 │
│  NovelDirector → PlotPlanner → Writer → Reviewers → Memory  │
└──────────────────┬──────────────────┬──────────────┬────────┘
                   │                  │              │
         ┌─────────▼─────────┐  ┌────▼────────┐  ┌─▼──────────┐
         │  StoryUnit Arc    │  │ ChapterDebt │  │BriefValidator│
         │  Generator        │  │  Tracker    │  │              │
         │  (Director)       │  │  (Writer)   │  │  (Reviewer)  │
         └───────────────────┘  └─────────────┘  └──────────────┘
                   │                  │              │
         ┌─────────▼──────────────────▼──────────────▼────────┐
         │         NovelMemory / StructuredDB                  │
         │   (StoryUnits, ChapterDebts, Validation Reports)   │
         └────────────────────────────────────────────────────┘
```

### Design Principles

1. **Orthogonal Integration**: New systems augment existing agents without breaking them
2. **Graceful Degradation**: Missing arc/debt data doesn't block chapter generation
3. **Sync-First**: All operations remain synchronous (no async)
4. **Pydantic-Native**: All models use Pydantic for validation and serialization
5. **Storage Abstraction**: Data persists via existing StructuredDB/NovelMemory interfaces

```python
from __future__ import annotations
from uuid import uuid4
from typing import Literal
from pydantic import BaseModel, Field


class StoryUnit(BaseModel):
    """Mini-arc structure: 3-7 chapters with setup → escalation → climax → resolution"""

    arc_id: str = Field(
        default_factory=lambda: str(uuid4()),
        description="UUID"
    )
    volume_id: int = Field(..., ge=1, description="Parent volume number")
    name: str = Field(..., min_length=1, max_length=100, description="Arc name (e.g., '新生试炼篇')")
    chapters: list[int] = Field(..., min_length=3, max_length=7, description="Chapters in this arc")

    # Arc structure
    phase: Literal["setup", "escalation", "climax", "resolution"] = Field(
        "setup", description="Current phase of the arc"
    )

    # Key beats
    hook: str = Field(..., min_length=1, description="Opening hook that launches the arc")
    escalation_point: int = Field(..., ge=1, description="Chapter where tension peaks")
    turning_point: int = Field(..., ge=1, description="Chapter where direction changes")
    closure_method: str = Field(..., min_length=1, description="How the arc resolves")
    residual_question: str = Field(
        ..., min_length=1, description="Unresolved element that hooks into next arc"
    )

    # Metadata
    status: Literal["planning", "in_progress", "completed"] = Field(
        "planning", description="Arc completion status"
    )
    completion_rate: float = Field(
        0.0, ge=0.0, le=1.0, description="0.0-1.0, chapters completed / total chapters"
    )


class ArcBrief(BaseModel):
    """Planning brief for a story arc - generated by NovelDirector"""

    arc_id: str = Field(..., description="Reference to parent StoryUnit")

    # Core conflict
    central_conflict: str = Field(..., min_length=1, description="Main conflict for this arc")
    protagonist_goal: str = Field(..., min_length=1, description="What protagonist wants in this arc")
    antagonist_goal: str = Field("", description="What antagonist wants (if applicable)")

    # Stakes
    success_consequence: str = Field(..., min_length=1, description="What happens if protagonist succeeds")
    failure_consequence: str = Field(..., min_length=1, description="What happens if protagonist fails")

    # Structure
    setup_beats: list[str] = Field(default_factory=list, description="Key beats in setup phase")
    escalation_beats: list[str] = Field(default_factory=list, description="Key beats in escalation phase")
    climax_description: str = Field(..., min_length=1, description="How the climax unfolds")
    resolution_outcome: str = Field(..., min_length=1, description="Arc resolution result")
```

**Integration Points**:
- `ChapterOutline.arc_id: str | None` - New field linking chapter to arc

```python
from __future__ import annotations
from datetime import datetime, timezone
from uuid import uuid4
from typing import Literal
from pydantic import BaseModel, Field


class ChapterDebt(BaseModel):
    """Narrative obligation requiring future payoff"""

    debt_id: str = Field(
        default_factory=lambda: str(uuid4()),
        description="UUID"
    )
    source_chapter: int = Field(..., ge=1, description="Chapter where promise was made")
    created_at: str = Field(
        default_factory=lambda: datetime.now(timezone.utc).isoformat(),
        description="Creation timestamp ISO format"
    )

    # Debt classification
    type: Literal["must_pay_next", "pay_within_3", "long_tail_payoff"] = Field(
        ..., description="Urgency classification"
    )
    description: str = Field(..., min_length=1, description="What needs to be resolved")

    # Specific obligations
    character_pending_actions: list[str] = Field(
        default_factory=list, description="Character-specific obligations"
    )
    emotional_debt: str | None = Field(
        None, description="Unresolved emotional tension"
    )

    # Resolution tracking
    target_chapter: int | None = Field(
        None, ge=1, description="When it should be paid (if specified)"
    )
    status: Literal["pending", "fulfilled", "overdue", "abandoned"] = Field(
        "pending", description="Current debt status"
    )
    fulfilled_at: int | None = Field(
        None, ge=1, description="Chapter where debt was paid"
    )
    fulfillment_note: str | None = Field(
        None, description="How the debt was resolved"
    )

    # Escalation
    urgency_level: Literal["normal", "high", "critical"] = Field(
        "normal", description="Escalation level"
    )
    escalation_history: list[dict] = Field(
        default_factory=list, description="Log of urgency changes"
    )


class DebtExtractionResult(BaseModel):
    """Result of extracting debts from a chapter"""

    chapter_number: int = Field(..., ge=1)
    debts: list[ChapterDebt] = Field(default_factory=list)
    extraction_method: Literal["llm", "rule_based", "hybrid"] = Field(
        "llm", description="Method used for extraction"
    )
    confidence: float = Field(
        1.0, ge=0.0, le=1.0, description="Confidence in extraction quality"
    )


class DebtContext(BaseModel):
    """Formatted debt context for Writer prompt injection"""

    chapter_number: int = Field(..., ge=1, description="Target chapter")
    pending_debts: list[ChapterDebt] = Field(default_factory=list)
    overdue_debts: list[ChapterDebt] = Field(default_factory=list)
    critical_debts: list[ChapterDebt] = Field(default_factory=list)

    formatted_summary: str = Field(
        "", description="Human-readable summary for LLM prompt"
    )
    token_count: int = Field(0, ge=0, description="Estimated token count of summary")
```

**Integration Points**:
- Writer reads `DebtContext` before generating chapter
- QualityReviewer creates debts after validating brief fulfillment
- ObligationTracker manages debt lifecycle

## Service Layer

### ObligationTracker Service

**File**: `src/novel/services/obligation_tracker.py` (new file)

```python
"""Runtime service for debt lifecycle management"""

from __future__ import annotations
import logging
from datetime import datetime, timezone
from pathlib import Path

from src.novel.models.debt import ChapterDebt, DebtContext
from src.novel.storage.structured_db import StructuredDB

log = logging.getLogger("novel")


class ObligationTracker:
    """Manages debt lifecycle: creation, escalation, fulfillment, query"""

    def __init__(self, db: StructuredDB) -> None:
        self.db = db
        self._ensure_tables()

    def _ensure_tables(self) -> None:
        """Create debts table if not exists"""
        self.db.execute("""
            CREATE TABLE IF NOT EXISTS chapter_debts (
                debt_id TEXT PRIMARY KEY,
                source_chapter INTEGER NOT NULL,
                created_at TEXT NOT NULL,
                type TEXT NOT NULL,
                description TEXT NOT NULL,
                target_chapter INTEGER,
                status TEXT NOT NULL,
                fulfilled_at INTEGER,
                fulfillment_note TEXT,
                urgency_level TEXT NOT NULL,
                data JSON
            )
        """)

    def add_debt(self, debt: ChapterDebt) -> None:
        """Add a new debt to the tracker"""
        import json

        data_json = json.dumps({
            "character_pending_actions": debt.character_pending_actions,
            "emotional_debt": debt.emotional_debt,
            "escalation_history": debt.escalation_history,
        })

        self.db.execute(
            """
            INSERT INTO chapter_debts
            (debt_id, source_chapter, created_at, type, description,
             target_chapter, status, fulfilled_at, fulfillment_note, urgency_level, data)
            VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?)
            """,
            (
                debt.debt_id,
                debt.source_chapter,
                debt.created_at,
                debt.type,
                debt.description,
                debt.target_chapter,
                debt.status,
                debt.fulfilled_at,
                debt.fulfillment_note,
                debt.urgency_level,
                data_json,
            ),
        )
        log.info("添加债务: %s (类型=%s, 来源章节=%d)", debt.debt_id, debt.type, debt.source_chapter)

    def get_debts_for_chapter(self, chapter_num: int) -> list[ChapterDebt]:
        """Get all relevant debts for a chapter (pending + overdue only)"""
        import json

        rows = self.db.execute(
            """
            SELECT * FROM chapter_debts
            WHERE status IN ('pending', 'overdue')
            AND source_chapter < ?
            ORDER BY urgency_level DESC, source_chapter ASC
            """,
            (chapter_num,),
        ).fetchall()

        debts: list[ChapterDebt] = []
        for row in rows:
            data = json.loads(row["data"])
            debts.append(
                ChapterDebt(
                    debt_id=row["debt_id"],
                    source_chapter=row["source_chapter"],
                    created_at=row["created_at"],
                    type=row["type"],
                    description=row["description"],
                    character_pending_actions=data.get("character_pending_actions", []),
                    emotional_debt=data.get("emotional_debt"),
                    target_chapter=row["target_chapter"],
                    status=row["status"],
                    fulfilled_at=row["fulfilled_at"],
                    fulfillment_note=row["fulfillment_note"],
                    urgency_level=row["urgency_level"],
                    escalation_history=data.get("escalation_history", []),
                )
            )

        return debts

    def mark_debt_fulfilled(
        self, debt_id: str, chapter_num: int, note: str | None = None
    ) -> None:
        """Mark a debt as fulfilled"""
        self.db.execute(
            """
            UPDATE chapter_debts
            SET status = 'fulfilled', fulfilled_at = ?, fulfillment_note = ?
            WHERE debt_id = ?
            """,
            (chapter_num, note, debt_id),
        )
        log.info("债务 %s 已完成于第 %d 章", debt_id, chapter_num)

    def escalate_debts(self, current_chapter: int) -> int:
        """Escalate overdue debts, return count of escalations"""
        import json

        # Get all pending debts
        rows = self.db.execute(
            """
            SELECT * FROM chapter_debts WHERE status = 'pending'
            """
        ).fetchall()

        escalated_count = 0
        for row in rows:
            debt_type = row["type"]
            source_chapter = row["source_chapter"]
            chapters_elapsed = current_chapter - source_chapter

            should_escalate = False
            new_urgency = row["urgency_level"]

            # Escalation rules
            if debt_type == "must_pay_next" and chapters_elapsed >= 1:
                should_escalate = True
                new_urgency = "critical"
            elif debt_type == "pay_within_3" and chapters_elapsed >= 3:
                should_escalate = True
                new_urgency = "high"
            elif chapters_elapsed >= 5 and new_urgency == "normal":
                should_escalate = True
                new_urgency = "high"

            if should_escalate:
                data = json.loads(row["data"])
                history = data.get("escalation_history", [])
                history.append({
                    "chapter": current_chapter,
                    "old_urgency": row["urgency_level"],
                    "new_urgency": new_urgency,
                    "reason": f"{debt_type} 债务经过 {chapters_elapsed} 章未解决",
                })
                data["escalation_history"] = history

                self.db.execute(
                    """
                    UPDATE chapter_debts
                    SET urgency_level = ?, status = 'overdue', data = ?
                    WHERE debt_id = ?
                    """,
                    (new_urgency, json.dumps(data), row["debt_id"]),
                )
                escalated_count += 1
                log.warning(
                    "债务 %s 升级为 %s (来源: 第%d章, 当前: 第%d章)",
                    row["debt_id"], new_urgency, source_chapter, current_chapter
                )

        return escalated_count

    def get_summary_for_writer(self, chapter_num: int, max_tokens: int = 1000) -> str:
        """Generate formatted summary for Writer prompt"""
        debts = self.get_debts_for_chapter(chapter_num)

        if not debts:
            return ""

        critical = [d for d in debts if d.urgency_level == "critical"]
        high = [d for d in debts if d.urgency_level == "high"]
        normal = [d for d in debts if d.urgency_level == "normal"]

        lines = ["## 待解决的叙事债务\n"]

        if critical:
            lines.append("### 🔴 必须在本章解决：")
            for debt in critical[:3]:  # Limit to top 3
                lines.append(f"- 第{debt.source_chapter}章遗留：{debt.description}")

        if high:
            lines.append("\n### 🟡 近期需要解决：")
            for debt in high[:2]:
                lines.append(f"- 第{debt.source_chapter}章遗留：{debt.description}")

        if normal and len(lines) < 8:  # Only if space left
            lines.append("\n### 🟢 长线伏笔：")
            for debt in normal[:2]:
                lines.append(f"- 第{debt.source_chapter}章遗留：{debt.description}")

        summary = "\n".join(lines)

        # Token estimation (rough: 1 Chinese char ≈ 2 tokens)
        estimated_tokens = len(summary) * 2
        if estimated_tokens > max_tokens:
            # Truncate and add notice
            truncate_at = max_tokens // 2
            summary = summary[:truncate_at] + f"\n\n（已截断，完整债务列表共 {len(debts)} 项）"

        return summary

    def get_debt_statistics(self) -> dict:
        """Get debt statistics for monitoring"""
        stats = {}

        # Count by status
        for status in ["pending", "fulfilled", "overdue", "abandoned"]:
            count = self.db.execute(
                "SELECT COUNT(*) as cnt FROM chapter_debts WHERE status = ?",
                (status,),
            ).fetchone()["cnt"]
            stats[f"{status}_count"] = count

        # Average fulfillment time
        avg_time = self.db.execute(
            """
            SELECT AVG(fulfilled_at - source_chapter) as avg_time
            FROM chapter_debts WHERE status = 'fulfilled'
            """
        ).fetchone()["avg_time"]
        stats["avg_fulfillment_chapters"] = round(avg_time, 2) if avg_time else 0

        return stats
```

### BriefValidator Tool

**File**: `src/novel/tools/brief_validator.py` (new file)

```python
"""Validate chapter fulfillment of chapter_brief"""

from __future__ import annotations
import logging
from typing import Any

from src.llm.llm_client import LLMClient, LLMResponse
from src.novel.models.validation import BriefFulfillmentReport, BriefItemResult
from src.novel.utils import extract_json_from_llm

log = logging.getLogger("novel")


# Prompt templates
_VALIDATION_SYSTEM = """\
你是一位专业的网文质量审查员。你的任务是验证章节内容是否完成了大纲中规定的任务。

任务：
1. 阅读章节内容和任务书（chapter_brief）
2. 逐项检查是否完成
3. 给出明确的通过/失败判断

返回严格的 JSON 格式（不要添加任何额外文字）：
{
  "main_conflict_fulfilled": true/false,
  "payoff_delivered": true/false,
  "character_arc_step_taken": true/false,
  "foreshadowing_planted": [true, false, ...],
  "foreshadowing_collected": [true, true, ...],
  "end_hook_present": true/false,
  "item_results": [
    {
      "item_name": "main_conflict",
      "expected": "预期内容",
      "fulfilled": true,
      "evidence": "章节中的证据文本",
      "reason": "通过/失败原因"
    }
  ],
  "unfulfilled_items": ["item1", "item2"],
  "overall_pass": true/false
}
"""

_VALIDATION_USER = """\
## 章节任务书
{brief_formatted}

## 章节正文（仅检查是否完成任务，不评价文笔）
{chapter_text}

请验证上述任务是否在正文中完成。
"""


class BriefValidator:
    """Tool for validating chapter brief fulfillment"""

    def __init__(self, llm_client: LLMClient) -> None:
        self.llm = llm_client

    def validate_chapter(
        self,
        chapter_text: str,
        chapter_brief: dict,
    ) -> BriefFulfillmentReport:
        """Validate if chapter fulfilled its brief

        Args:
            chapter_text: Generated chapter text
            chapter_brief: ChapterOutline.chapter_brief dict

        Returns:
            BriefFulfillmentReport with per-item results
        """
        if not chapter_brief:
            log.warning("章节任务书为空，跳过验证（默认通过）")
            return BriefFulfillmentReport(
                chapter_number=0,  # Will be set by caller
                overall_pass=True,
                pass_rate=1.0,
            )

        # Format brief for LLM
        brief_lines = []
        if chapter_brief.get("main_conflict"):
            brief_lines.append(f"- 主冲突：{chapter_brief['main_conflict']}")
        if chapter_brief.get("payoff"):
            brief_lines.append(f"- 本章爽点/回报：{chapter_brief['payoff']}")
        if chapter_brief.get("character_arc_step"):
            brief_lines.append(f"- 角色弧线推进：{chapter_brief['character_arc_step']}")

        plant_list = chapter_brief.get("foreshadowing_plant", [])
        if isinstance(plant_list, str):
            plant_list = [plant_list]
        if plant_list:
            brief_lines.append(f"- 需要埋设的伏笔：{'、'.join(plant_list)}")

        collect_list = chapter_brief.get("foreshadowing_collect", [])
        if isinstance(collect_list, str):
            collect_list = [collect_list]
        if collect_list:
            brief_lines.append(f"- 需要回收的伏笔：{'、'.join(collect_list)}")

        if chapter_brief.get("end_hook_type"):
            brief_lines.append(f"- 章尾钩子类型：{chapter_brief['end_hook_type']}")

        if not brief_lines:
            log.warning("章节任务书无有效字段，跳过验证（默认通过）")
            return BriefFulfillmentReport(
                chapter_number=0,
                overall_pass=True,
                pass_rate=1.0,
            )

        brief_formatted = "\n".join(brief_lines)

        # Truncate chapter text if too long (keep first 3000 chars)
        chapter_excerpt = chapter_text[:3000]
        if len(chapter_text) > 3000:
            chapter_excerpt += "\n\n（章节过长，已截取前 3000 字符）"

        user_msg = _VALIDATION_USER.format(
            brief_formatted=brief_formatted,
            chapter_text=chapter_excerpt,
        )

        messages = [
            {"role": "system", "content": _VALIDATION_SYSTEM},
            {"role": "user", "content": user_msg},
        ]

        try:
            response: LLMResponse = self.llm.chat(
                messages, temperature=0.3, json_mode=True, max_tokens=2000
            )
            parsed = extract_json_from_llm(response.content)

            # Build report from LLM response
            report = BriefFulfillmentReport(
                chapter_number=0,  # Will be set by caller
                main_conflict_fulfilled=parsed.get("main_conflict_fulfilled", True),
                payoff_delivered=parsed.get("payoff_delivered", True),
                character_arc_step_taken=parsed.get("character_arc_step_taken", True),
                foreshadowing_planted=parsed.get("foreshadowing_planted", []),
                foreshadowing_collected=parsed.get("foreshadowing_collected", []),
                end_hook_present=parsed.get("end_hook_present", True),
                item_results=[
                    BriefItemResult(**item) for item in parsed.get("item_results", [])
                ],
                unfulfilled_items=parsed.get("unfulfilled_items", []),
                overall_pass=parsed.get("overall_pass", True),
            )

            # Calculate pass rate
            total_items = len(report.item_results) if report.item_results else 1
            passed_items = sum(1 for item in report.item_results if item.fulfilled)
            report.pass_rate = passed_items / total_items

            # Generate suggested debts for unfulfilled mandatory items
            if not report.main_conflict_fulfilled:
                report.suggested_debts.append(f"主冲突未解决: {chapter_brief.get('main_conflict')}")
            if not report.payoff_delivered and chapter_brief.get("payoff"):
                report.suggested_debts.append(f"爽点未兑现: {chapter_brief.get('payoff')}")

            log.info(
                "章节任务书验证完成: 总体通过=%s, 通过率=%.2f",
                report.overall_pass, report.pass_rate
            )

            return report

        except Exception as exc:
            log.error("章节任务书验证失败: %s", exc)
            # Return permissive default on error
            return BriefFulfillmentReport(
                chapter_number=0,
                overall_pass=True,
                pass_rate=1.0,
```

```python
"""Extract narrative debts from generated chapters"""

from __future__ import annotations
import logging
import re
from typing import Any

from src.llm.llm_client import LLMClient, LLMResponse
from src.novel.models.debt import ChapterDebt, DebtExtractionResult
from src.novel.utils import extract_json_from_llm

log = logging.getLogger("novel")


_EXTRACTION_SYSTEM = """\
你是一位专业的网文叙事顾问。你的任务是从章节内容中识别出所有需要后续解决的叙事债务（narrative debts）。

叙事债务包括：
1. 角色明确的承诺或计划（"我会..."/"明天我..."/"下次..."）
2. 引入但未解决的冲突（打斗被打断、悬念未揭晓）
3. 对读者的暗示（"这件事的后果，他还不知道..."）
4. 未完成的角色行动（"他转身离开，准备去..."）
5. 情感未解决（"她心中的疑惑更深了..."）

返回严格的 JSON 格式（不要添加任何额外文字）：
{
  "debts": [
    {
      "type": "must_pay_next" | "pay_within_3" | "long_tail_payoff",
      "description": "具体债务描述",
      "character_pending_actions": ["角色需要做的事"],
      "emotional_debt": "情感债务描述" | null,
      "target_chapter": null | 具体章节号,
      "urgency_level": "normal" | "high" | "critical"
    }
  ]
}
"""

_EXTRACTION_USER = """\
## 章节正文
{chapter_text}

请识别出所有需要后续解决的叙事债务。
"""


class DebtExtractor:
    """Service for extracting debts from generated chapters"""

    def __init__(self, llm_client: LLMClient | None = None) -> None:
        self.llm = llm_client

    def extract_from_chapter(
        self,
        chapter_text: str,
        chapter_number: int,
        method: str = "hybrid",
    ) -> DebtExtractionResult:
        """Extract debts from chapter text

        Args:
            chapter_text: Generated chapter content
            chapter_number: Chapter number (for source_chapter)
            method: "llm", "rule_based", or "hybrid"

        Returns:
            DebtExtractionResult with extracted debts
        """
        if method == "rule_based":
            return self._extract_rule_based(chapter_text, chapter_number)
        elif method == "llm" and self.llm:
            return self._extract_llm(chapter_text, chapter_number)
        elif method == "hybrid":
            # Combine both approaches
            rule_result = self._extract_rule_based(chapter_text, chapter_number)
            if self.llm:
                llm_result = self._extract_llm(chapter_text, chapter_number)
                # Merge debts (LLM has priority)
                combined_debts = llm_result.debts + [
                    d for d in rule_result.debts
                    if not any(
                        self._is_similar_debt(d, ld) for ld in llm_result.debts
                    )
                ]
                return DebtExtractionResult(
                    chapter_number=chapter_number,
                    debts=combined_debts,
                    extraction_method="hybrid",
                    confidence=0.85,
                )
            return rule_result
        else:
            log.warning("LLM 不可用，回退到规则提取")
            return self._extract_rule_based(chapter_text, chapter_number)

    def _extract_rule_based(
        self, chapter_text: str, chapter_number: int
    ) -> DebtExtractionResult:
        """Rule-based extraction using regex patterns"""
        debts: list[ChapterDebt] = []

        # Pattern 1: Explicit promises ("我会...", "明天我...")
        promise_patterns = [
            r'["""](我|他|她).{0,10}(会|要|必须|一定).{5,30}["""]',
            r'["""](明天|下次|稍后|之后).{5,30}["""]',
        ]

        for pattern in promise_patterns:
            matches = re.finditer(pattern, chapter_text)
            for match in matches:
                text = match.group(0)
                debts.append(
                    ChapterDebt(
                        source_chapter=chapter_number,
                        type="pay_within_3",
                        description=f"角色承诺: {text}",
                        urgency_level="normal",
                    )
                )

        # Pattern 2: Unresolved tension ("心中的疑惑", "还不知道")
        tension_patterns = [
            r'(疑惑|疑问|好奇|不解).{0,20}(更深|加深|涌起)',
            r'(还不知道|不知道的是|殊不知)',
        ]

        for pattern in tension_patterns:
            matches = re.finditer(pattern, chapter_text)
            for match in matches:
                text = match.group(0)
                debts.append(
                    ChapterDebt(
                        source_chapter=chapter_number,
                        type="long_tail_payoff",
                        description=f"情感悬念: {text}",
                        emotional_debt=text,
                        urgency_level="normal",
                    )
                )

        # Pattern 3: Interrupted actions ("转身离开", "准备去")
        action_patterns = [
            r'(转身|起身|迈步).{0,10}(离开|前往|走向)',
            r'准备(去|做|开始)',
        ]

        for pattern in action_patterns:
            matches = re.finditer(pattern, chapter_text)
            for match in matches:
                text = match.group(0)
                debts.append(
                    ChapterDebt(
                        source_chapter=chapter_number,
                        type="must_pay_next",
                        description=f"待完成动作: {text}",
                        character_pending_actions=[text],
                        urgency_level="high",
                    )
                )

        # Deduplicate similar debts
        debts = self._deduplicate_debts(debts)

        log.info("规则提取识别到 %d 个债务", len(debts))

        return DebtExtractionResult(
            chapter_number=chapter_number,
            debts=debts,
            extraction_method="rule_based",
            confidence=0.6,
        )

    def _extract_llm(
        self, chapter_text: str, chapter_number: int
    ) -> DebtExtractionResult:
        """LLM-based extraction"""
        if not self.llm:
            raise ValueError("LLM 客户端未初始化")

        # Truncate if too long
        chapter_excerpt = chapter_text[:4000]
        if len(chapter_text) > 4000:
            chapter_excerpt += "\n\n（章节过长，已截取前 4000 字符）"

        user_msg = _EXTRACTION_USER.format(chapter_text=chapter_excerpt)

        messages = [
            {"role": "system", "content": _EXTRACTION_SYSTEM},
            {"role": "user", "content": user_msg},
        ]

        try:
            response: LLMResponse = self.llm.chat(
                messages, temperature=0.5, json_mode=True, max_tokens=2000
            )
            parsed = extract_json_from_llm(response.content)

            debts: list[ChapterDebt] = []
            for debt_data in parsed.get("debts", []):
                debts.append(
                    ChapterDebt(
                        source_chapter=chapter_number,
                        type=debt_data.get("type", "pay_within_3"),
                        description=debt_data.get("description", "未描述"),
                        character_pending_actions=debt_data.get(
                            "character_pending_actions", []
                        ),
                        emotional_debt=debt_data.get("emotional_debt"),
                        target_chapter=debt_data.get("target_chapter"),
                        urgency_level=debt_data.get("urgency_level", "normal"),
                    )
                )

            log.info("LLM 提取识别到 %d 个债务", len(debts))

            return DebtExtractionResult(
                chapter_number=chapter_number,
                debts=debts,
                extraction_method="llm",
                confidence=0.9,
            )

        except Exception as exc:
            log.error("LLM 提取失败: %s", exc)
            return DebtExtractionResult(
                chapter_number=chapter_number,
                debts=[],
                extraction_method="llm",
                confidence=0.0,
            )

    @staticmethod
    def _is_similar_debt(debt1: ChapterDebt, debt2: ChapterDebt) -> bool:
        """Check if two debts are similar (for deduplication)"""
        # Simple similarity: same type and overlapping description
        if debt1.type != debt2.type:
            return False

        desc1_words = set(debt1.description.split())
        desc2_words = set(debt2.description.split())

        if not desc1_words or not desc2_words:
            return False

        overlap = len(desc1_words & desc2_words)
        union = len(desc1_words | desc2_words)

        similarity = overlap / union if union > 0 else 0
        return similarity > 0.5

    @staticmethod
    def _deduplicate_debts(debts: list[ChapterDebt]) -> list[ChapterDebt]:
        """Remove duplicate debts"""
        unique: list[ChapterDebt] = []
        for debt in debts:
            if not any(
                DebtExtractor._is_similar_debt(debt, u) for u in unique
            ):
                unique.append(debt)
        return unique
```

## Performance Considerations

### Token Usage Optimization

1. **BriefValidator**: Use chapter excerpt (first 3000 chars) instead of full text
2. **DebtExtractor**: Hybrid mode - rule-based first, LLM only for complex cases
3. **Debt Context**: Cap at 1000 tokens, prioritize critical debts
4. **Foreshadowing**: Use BM25 retrieval before LLM analysis (Phase 2)

### Database Performance

1. **Indexes**: Add indexes on `status`, `source_chapter`, `urgency_level`
2. **Query Optimization**: Single query for all relevant debts per chapter
3. **Batch Operations**: Escalate debts in batch per chapter
4. **Connection Pooling**: Reuse StructuredDB connections

### Memory Management

1. **Debt Loading**: Load only pending/overdue debts, not historical
2. **Context Window**: Truncate debt summary if exceeds max_tokens
3. **Cleanup**: Archive fulfilled debts older than 10 chapters
